## Comment évaluer un développeur Next.js

### Critères d'évaluation

| Critère | Poids | Comment vérifier |
|---------|-------|------------------|
| Expérience Next.js | 30% | Portfolio, questions techniques |
| Qualité du code | 25% | GitHub, test technique |
| Communication | 20% | Entretien, réactivité |
| Compréhension besoin | 15% | Questions posées |
| Références | 10% | Appeler anciens clients |

### Questions techniques à poser

**Niveau junior/confirmé :**
1. Quelle est la différence entre `pages/` et `app/` router ?
2. Quand utiliser `getStaticProps` vs `getServerSideProps` ?
3. Comment optimiser les images avec Next.js ?
4. Qu'est-ce qu'un Server Component vs Client Component ?

**Niveau senior/expert :**
1. Comment implémenter l'ISR avec revalidation on-demand ?
2. Expliquez le pattern Streaming avec Suspense dans Next.js
3. Comment architecturer une app multi-tenant avec Next.js ?
4. Quelles stratégies de cache utilisez-vous ?
5. Comment gérez-vous l'état entre Server et Client Components ?

### Red flags à éviter

| Signal d'alarme | Pourquoi c'est grave |
|-----------------|---------------------|
| Pas de GitHub/portfolio | Impossible de vérifier |
| Ne connaît que `pages/` router | Stack obsolète |
| Pas de TypeScript | Pratique non professionnelle |
| Ne mentionne jamais la performance | Manque de rigueur |
| Questions vagues sur les besoins | Manque d'expérience projet |

### Green flags encourageants

| Bon signe | Ce que ça indique |
|-----------|------------------|
| Contributions open-source | Passion et compétence |
| Blog technique | Veille et pédagogie |
| Questions pertinentes | Expérience projet |
| Mentionne Vercel/Edge | À jour sur l'écosystème |
| Parle performance spontanément | Mindset professionnel |

## Processus de recrutement recommandé

### Étape 1 : Définir le besoin (1-2 jours)

- Type de projet et stack technique
- Budget et délais
- Niveau d'expérience requis
- Mode de travail (remote, présentiel, hybride)

### Étape 2 : Sourcing (1 semaine)

- Publier sur 2-3 plateformes
- Activer son réseau (LinkedIn, recommandations)
- Contacter des agences si budget > 10k€

### Étape 3 : Pré-qualification (3-5 jours)

- Analyser portfolios et GitHub
- Appel de 15-20 min pour valider le fit
- Short-list de 3-5 candidats

### Étape 4 : Évaluation approfondie (1 semaine)

- Entretien technique (45-60 min)
- Test technique si pertinent (2-4h max)
- Vérification des références

### Étape 5 : Décision et contractualisation (3-5 jours)

- Comparaison des propositions
- Négociation si nécessaire
- Signature du contrat/devis
